YouTube Gaming Trend Analyzer

A Python-based data pipeline that pulls trending gaming videos from YouTube, matches games using PostgreSQL aliases, tracks unknown terms, and stores trend results over time.

Python PostgreSQL YouTube API Data Pipeline

Project Overview

This project turns unstructured YouTube gaming video titles into structured trend data. It uses a database-driven game alias system to identify games, count mentions, and store results for historical tracking.

Features

  • YouTube Data API integration
  • PostgreSQL database storage
  • Database-driven game alias matching
  • Unknown term tracking for game discovery
  • Automated trend collection
  • Historical trend tracking with timestamps

Pipeline

System data flow:

YouTube API
   ↓
Python Script
   ↓
PostgreSQL Database
   ├── games
   ├── game_aliases
   ├── trending_games
   └── unknown_terms
